# Interactive 3D Portfolio

An immersive 3D room experience showcasing my skills, projects, and personality through interactive objects and mini-games.

---

## Features

### **Explorable Room**
Navigate a fully 3D environment with multiple interactive zones:
- **Desk Area** - View my projects and skills via the laptop interface
- **Gaming Corner** - Play retro games on the TV (Snake, Pong, Tetris, Space Invaders)
- **Certificate Board** - View professional certifications
and more..

### **Interactive Elements**
- Smooth camera transitions between zones
- Hover effects with glowing outlines
- Click-to-explore object interactions
- Typewriter dialogue system
- Mobile touch controls for games

### **Built-in Games**
Classic arcade games rendered on a 3D TV screen:
- Snake with progressive difficulty
- Pong with AI opponent
- Tetris with score tracking
- Space Invaders with wave progression

---

## Tech Stack

- **Three.js** - 3D rendering and scene management
- **ONNX Runtime** - Neural network chess AI
- **Vanilla JavaScript** - Core logic and interactions
- **GLB/GLTF** - 3D model format
- **Web Audio API** - Background music system
